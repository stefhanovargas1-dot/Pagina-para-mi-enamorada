# Galería de Fotos - Mi Amorcitooo

Una galería de fotos interactiva con efecto deslizante.

## Cómo ver la página

Para ver esta página correctamente con todas las animaciones, necesitas ejecutar un servidor local.

### Opción con Python (Recomendada)

1. Abre una terminal (consola) en esta carpeta.
2. Escribe el siguiente comando y presiona Enter:

   ```bash
   python -m http.server 8080
   ```

3. Abre tu navegador web y entra a: [http://localhost:8080](http://localhost:8080)

## Estructura

- `index.html`: Estructura de la página.
- `style.css`: Estilos y animaciones.
- `script.js`: Lógica para el funcionamiento del slider.
